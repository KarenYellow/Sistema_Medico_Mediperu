Mediperu es un software que se utilizará en el ámbito de la salud para la gestión de informaciónclínica, administrativa y financiera. Este tipo de sistema será diseñado para hospitales,clínicas, consultorios médicos, laboratorios y otros establecimientos de atención médica para mejorar la calidad de a atención y la eficiencia en la gestión de los procesos.
